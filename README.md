Nombre del proyecto: Tienda diagnostico

Descripción:
Registra productos con  Node.js y  SQLite.

Tecnologías utilizadas: node.js, react, sqlite, express

Funcionalidades:
CRUD de productos, búsqueda, ordenamiento y edición desde la interfaz.

Instrucciones para ejecutar el proyecto:
1 Abrir una terminal dentro de la carpeta del proyecto.
2 Ejecutar node app.js.
3 Abrir http://localhost:3000 en el navegador.

La base de datos se crea automáticamente como 'database.sqlite' la primera vez que se inicia el servidor.

Indicación de si se usó IA y para qué: se usó IA para toda la base del proyecto y revisión de errores.
